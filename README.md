# Payment Stuff

This is the payment module I was working on. It lets you take credit cards and bank info. 

### How to run it:
1. Open the folder in your terminal.
2. Type `npm install` to get the stuff it needs.
3. Type `npm run dev` to start it up.
4. Go to `http://localhost:5001` in your browser.

### What's inside:
- It uses React and Vite.
- You can put it into other projects if you want.
- I added some fake data so you can see how it looks.

Let me know if you have any questions!
